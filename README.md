# Pet Blog

## How to start the project
Open command line.<br/>

#### For Linux
Enter *./gradlew bootRun*.

#### For Windows
Enter *gradlew bootRun*.

### The project will be run.

## How to test the project 
Open command line.<br/>
#### For Linux
Enter *./gradlew test*.
#### For Windows
Enter *gradlew test*.

### The tests will be run.
